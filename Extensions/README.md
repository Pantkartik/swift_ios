# Extensions in Swift 🧩

Extensions add new functionality to an existing class, structure, enumeration, or protocol type. This includes the ability to extend types for which you don't have access to the original source code (known as retroactive modeling).

## 📄 Contents

- `Extensions_Basics.swift`: Demonstrates adding computed properties, methods, and initializers to existing types (like `String` or `Int`).

## 💡 Key Concepts
- **Adding Computed Properties**: Extensions can add new computed properties, but not stored properties.
- **Protocol Conformance**: You can use extensions to make an existing type conform to a protocol.
- **Nested Types**: Adding new nested types to existing classes, structures, or enumerations.
