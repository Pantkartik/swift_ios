# Networking in Swift 🌐

This directory contains resources and examples for making network requests in Swift and iOS applications.

## Overview
Networking is a fundamental part of most modern apps, enabling them to communicate with servers to fetch data, authenticate users, and more. Swift provides robust APIs for handling network operations securely and efficiently.

## Core Concepts
- **URLSession**: The foundation for all networking in iOS, handling data transfer tasks.
- **REST APIs**: Interacting with RESTful web services using GET, POST, PUT, and DELETE methods.
- **JSON Parsing**: Encoding and decoding data using `Codable` and `JSONDecoder`/`JSONEncoder`.
- **Concurrency**: Using `async/await` and Combine for asynchronous network calls.
- **Error Handling**: Managing network failures, invalid responses, and parsing errors.

## Advanced Topics
- **Authentication**: Implementing OAuth, JWT, and session management.
- **WebSockets**: Real-time bidirectional communication.
- **Background Downloads**: Downloading large files while the app is in the background.
- **Security**: Certificate pinning and ATS (App Transport Security).
