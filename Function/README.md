# Functions in Swift ⚙️

Functions are self-contained chunks of code that perform a specific task. You give a function a name that identifies what it does, and this name is used to "call" the function to perform its task when needed.

## 📄 Contents

- `function.swift`: Basic function declaration and usage.
- `reverse_string.swift`: Example of a function solving a common algorithm.
- `question1.swift`, `question2.swift`, `question3.swift`: Various functional exercises and examples.

## 💡 Key Concepts
- **Argument Labels and Parameter Names**: Enhancing readability with external argument labels.
- **In-Out Parameters**: Allowing a function to modify a variable passed as an argument.
- **Return Types**: Functions can return single values, tuples, or even be discarded (`@discardableResult`).
