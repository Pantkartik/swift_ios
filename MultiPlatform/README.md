# Multi-Platform Development in Swift 🌐

A comprehensive guide to extending iOS applications to macOS and visionOS.

## 🖥️ macOS Development (Catalyst & Native)
- `macOS_Guide.swift`: Explores building native Mac apps and utilizing Mac Catalyst to bring iOS apps to macOS. Covers menu bars, window management, and pointer interactions.

## 🥽 visionOS Development
- `visionOS_Guide.swift`: Concepts for Spatial Computing. Covers volumetric windows, 3D content, and immersive spaces using SwiftUI and RealityKit.

## 🛠️ Shared Codebase Strategies
Learn how to structure modules to share business logic (Combine, SwiftData, Networking) while providing platform-specific UI layers.
