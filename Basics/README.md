# Swift Basics 🍎

This directory contains the foundational concepts of the Swift programming language.

## 📚 Topics Covered

### 1. Variables and Constants
- `let` vs `var`.
- Type inference and explicit types.

### 2. Control Flow
- `if`, `else`, `switch` statements.
- `for-in` and `while` loops.

### 3. Collection Types
- Arrays, Dictionaries, and Sets.

### 4. Basic Features
- Optionals (`if let`, `guard let`, nil coalescing).
- Functions and Closures.

## 🚀 Getting Started
Start with the subdirectories `01_Basics` and `02_Basics` for a step-by-step introduction.
Check `Advanced_Swift_Features.swift` once you are comfortable with the core syntax.
