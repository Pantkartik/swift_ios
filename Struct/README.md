# Structures in Swift 🏗️

Structures are general-purpose, flexible constructs that become the building blocks of your program's code. You define properties and methods to add functionality to your structures using the exact same syntax as you use to define constants, variables, and functions.

## 📄 Contents

- `struct.swift`: Basic implementation and usage of structures in Swift.

## 💡 Key Concepts
- **Value Types**: Structures are value types, meaning they are copied when passed around, ensuring safety and predictability.
- **Memberwise Initializers**: Swift automatically provides memberwise initializers for structures if you don't define a custom one.
- **Mutating Methods**: Methods that modify the structure's properties must be marked with the `mutating` keyword.
