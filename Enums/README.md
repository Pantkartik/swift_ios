# Enumerations in Swift 🏷️

An enumeration defines a common type for a group of related values and enables you to work with those values in a type-safe way within your code.

## 📄 Contents

- `Enums_Advanced.swift`: Demonstrates advanced enumeration usage including associated values, raw values, and recursive enumerations.

## 💡 Key Concepts
- **Associated Values**: Store values of any given type alongside the case value.
- **Raw Values**: Prepopulate enum cases with default values (like strings or integers).
- **CaseIterable**: A protocol that provides a collection of all of an enum's cases.
- **Indirect Enums**: Enums that can be recursive, marked with the `indirect` keyword.
