# Swift Concurrency ⚡️

Mastering structured concurrency and safe state management in asynchronous code.

## 🧵 Core Concepts

### Async / Await
The modern way to write asynchronous code that reads like synchronous code.
- Handling sequential and parallel asynchronous tasks.

### Tasks & Task Groups
Managing the lifecycle of asynchronous operations.
- Cancellation and error propagation.

### Actors
Ensuring thread-safety by protecting mutable state.
- `MainActor` for UI updates.
- Custom actors for background processing.

## 📂 Examples
- Check the `.swift` files in this directory for implementations of these patterns.
