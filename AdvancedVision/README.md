# Advanced Vision Tasks 👁️

Deep dive into complex image processing and spatial understanding using Apple's Vision framework and related technologies.

## 🎯 Topics Covered

- **Custom Model Integration**: Integrating CoreML models with Vision requests for specialized object detection.
- **Body & Hand Pose Detection**: `VNDetectHumanBodyPoseRequest` and `VNDetectHumanHandPoseRequest` for gesture recognition.
- **Trajectory Detection**: Tracking moving objects over time in a video stream.
- **3D Spatial Tracking**: Basics for ARKit and Vision integration.

## 📄 Files
- `Vision_Pro_Concepts.swift`: Demonstrates advanced Vision framework requests.
