# Swift & iOS Learning Path 🚀

Welcome to my personal collection of Swift and iOS development resources, examples, and experiments. This repository is structured to cover everything from the absolute basics to advanced topics like AI integration and complex concurrency.

## 📌 Overview

This project serves as a living documentation of my journey through the Swift ecosystem. It includes code snippets, small projects, and deep dives into specific frameworks.

## 📂 Repository Structure

The repository is organized by topic. Here's a high-level overview:

- **[Basics](./Basics)**: Variables, Types, Control Flow, and Swift fundamentals.
- **[Advanced Swift](./Advanced_Swift)**: Property Wrappers, Result Builders, and Generics.
- **[SwiftUI](./SwiftUI)**: Modern UI development with declarative syntax.
- **[Concurrency](./Concurrency)**: Async/await, Actors, and Structured Concurrency.
- **[AI Integration](./AI_Integration)**: CoreML, Vision Framework, and Local LLMs.
- **[Networking](./Networking)**: URLSession, REST APIs, and Combine.
- **[Persistence](./Persistence)**: SwiftData and Core Data.
- **[Testing](./Testing)**: Unit Testing, Mocking, and UI Testing.

For a detailed breakdown, see [STRUCTURE.md](./STRUCTURE.md).

## 🛠️ Getting Started

### Prerequisites
- **Xcode**: 15.0+ recommended.
- **Swift**: 5.9+ recommended.
- **macOS**: Sonoma 14.0+ recommended.

### How to use this repo
1. Clone the repository.
2. Open the `.swift` files in Xcode.
3. Most files are designed as standalone scripts or SwiftUI Views that can be previewed or run in a Playground.
